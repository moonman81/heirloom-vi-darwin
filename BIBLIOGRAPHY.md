# BIBLIOGRAPHY — heirloom-vi-darwin

Primary sources for the ex/vi lineage.

## Historical papers

- Joy, William N., Mark Horton (1979). *An Introduction to Display
  Editing with Vi*. UC Berkeley Computer Science Department.
  The founding vi paper.
- Coulouris, George F. (1976). *Editing Text on a Video Display*.
  Queen Mary College, University of London. Foundation of `em`
  which influenced Joy's design.
- Bourne, Stephen R. (1978). *The UNIX Shell*. Bell System Technical
  Journal 57(6). Context for the ex `!` shell-out mechanism.

## Books

- Lamb, Linda (1990). *Learning the vi Editor*. O'Reilly.
- Robbins, Arnold, Elbert Hannah, Linda Lamb (2008). *Learning the
  vi and vim Editors, 7th ed*. O'Reilly.
- Kernighan, Brian W., Rob Pike (1984). *The UNIX Programming
  Environment*. Prentice-Hall. Contextualises the ed-ex-vi lineage.

## Standards

- POSIX.1-2001 (IEEE Std 1003.1-2001), Shell and Utilities §ex, §vi.
- Single UNIX Specification Version 3 §ex, §vi.

## Distributions

- 2.11BSD tape distribution (June 1988) — the specific vintage of
  ex/vi 3.7 (6/7/85) that Ritter's tarball descends from.
- 1BSD (March 1978), 2BSD (May 1979), 3BSD (September 1979) — see
  `moonman81/heirloom-ancestors-darwin` for extracted reference copies.

## Ritter's port README

- Ritter, Gunnar (2000). "README.NEW" inside `ritter_vi.tar.gz`.
  Discusses the ISO character-set additions and portability
  disclaimers. Public copy at
  <https://www.tuhs.org/Archive/Applications/Ritter_Vi/2000/README.NEW>.
